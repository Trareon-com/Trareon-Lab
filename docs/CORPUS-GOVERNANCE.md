# Corpus Governance

## Classes

| Class | Path | Redistribution |
|---|---|---|
| public | `validation/public/` | redistributable fixtures only |
| synthetic | `validation/synthetic/` | generated; recipe required |
| restricted | `validation/restricted/` | no public redistribution; access controlled |

## Fixture requirements

Every fixture records provenance, license/legal basis, sensitivity, expected result, hashes, generation recipe when synthetic, retention, access control, and redistribution rule.

Real case evidence may never be copied into the public corpus.
