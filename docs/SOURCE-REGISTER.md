# Trareon Lab Controlled Source Register

Register ini mengendalikan metadata sumber, bukan isi normatif berhak cipta. Setiap entry mencatat title, issuing authority, exact edition/version, publication/effective date, Indonesian adoption status, authoritative locator, controlled-copy owner, access date, applicability, license/copyright handling, dan next review date. Salinan standar berlisensi tetap berada pada penyimpanan terkendali milik organisasi dan tidak dimasukkan ke repository ini. Metadata katalog atau ringkasan publik tidak menjadi dasar klaim sertifikasi, akreditasi, conformity, atau admissibility.

## Control rules

- Official publisher or government locators are the only basis for normative status metadata.
- A catalogue page is not a substitute for a licensed standard when clause-level mapping is required.
- Every locally retained controlled copy must record its digest, acquisition date, access classification, owner, and supersession state outside this public planning register.
- Source updates trigger impact review against the traceability index, affected ADRs, validation evidence, documentation, and product wording.
- Standards and regulatory controls describe supportable processes; they do not establish that Trareon Lab is certified, an organization is accredited, or evidence is automatically admissible.
