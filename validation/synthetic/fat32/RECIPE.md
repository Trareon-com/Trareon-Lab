# Synthetic FAT32 corpus (fat32-synth-v1)

| Field | Value |
|---|---|
| Class | synthetic |
| Format | `TRFSFA32` (`*.trfsfa`) |
| Purpose | Day 17 FAT32 enumeration semantics |
| Redistribution | allowed |
| Sensitivity | none |

## Recipe

Same layout as NTFS synth with cluster IDs (root = `0`) and POSIX-style `/` paths.

Demo: `validation/synthetic/fat32/fat32-synth-demo.trfsfa` (regenerated by `cargo test -p lab-fs --test fat_enumerate`).

## Non-claims

Not a full FAT boot/FAT table parser.
